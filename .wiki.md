# Project Summary
MVB Pro is a cutting-edge SaaS platform tailored for developers and businesses to manage and deploy chatbots with integrated real-time trading capabilities. This platform significantly enhances customer engagement through automated interactions and sophisticated trading functionalities, allowing users to effortlessly toggle between demo and live trading accounts while effectively configuring their bots and managing devices and licenses.

# Project Module Description
The project comprises a React frontend and a Node.js backend. Key functionalities include:
- Robust user authentication and management
- Advanced project management tools
- A UI component library for chat interfaces
- A notification system utilizing Sonner for toast messages
- MySQL database integration for data management
- A comprehensive dashboard for monitoring bot activity and trading statistics
- Enhanced bot interface for configuring trading strategies
- Real trading system integrated with the Deriv API
- Improved bot analysis logic to prevent infinite loops and enhance performance
- Device management with a limit of two devices per user
- Continuous data collection and analysis for informed trading decisions
- A sophisticated license management system with validation and device fingerprinting
- Automatic bot activation upon WebSocket connection, independent of Deriv balance

# Usage
1. **Install Dependencies**:
   - Run `npm install` or `pnpm install` in both the `shadcn-ui` and `shadcn-ui/server` directories.
2. **Configure Backend**:
   - Navigate to the server directory and run `npm install`.
   - Create a `.env` file with your MySQL credentials:
     ```
     DB_HOST=localhost
     DB_USER=root
     DB_PASSWORD=your_password
     DB_NAME=bot_mvb_pro
     JWT_SECRET=your_jwt_secret_here
     ```
   - Execute the database setup script: `node scripts/setup-database.js`.
   - Start the server: `npm run dev`.
3. **Start Frontend**:
   - In another terminal, run `npm run dev` or `pnpm run dev` from the root directory.
